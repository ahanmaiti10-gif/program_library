# Program Library

This is my first official coding project made with Python. This is a library that contains several elementary programs such as:

 1. A modulated 'Hello world' program.
 2. A capacity calculator that calculates the true storage capacity of a device based on the manufacturer's advertised capacity.
 3. A simple number guessing game where the user selects their own bounds.
 4. A login game with various difficulty levels.
 5. A word guessing game where the user has to guess a randomly selected word from a list of words.
 6. A probability simulator that simulates coin flips, dice rolls, and card draws to demonstrate the concept of probability.
 7. A rock, paper, scissors game where the user plays against the computer.
 
 And hopefully many more!
 
 All of these programs are designed to be simple and easy to understand, making them perfect for beginners who are just starting to learn how to code. This program library is a great resource for anyone looking to learn how to code in Python and have fun while doing it!
